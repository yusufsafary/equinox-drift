# Equinox Drift

A mobile-first, browser-based 3D horse-riding exploration game built with Three.js.

Ride through surreal biomes where physics, light, and reality bend in strange, beautiful ways.

## Play

**Live:** https://yusufsafary.github.io/equinox-drift/

## Features

- Free-roam 3D world with 3 distinct surreal biomes
- Mobile-first virtual joystick controls
- Keyboard support (WASD / arrows)
- Drift Shards to collect
- Equinox Cycle sky animations
- Ghost trail sprint effects

## Tech Stack

- Three.js (3D rendering)
- cannon-es (physics)
- React + Vite
- Framer Motion (animations)
- TypeScript

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
BASE_PATH=/equinox-drift/ NODE_ENV=production pnpm build
```
